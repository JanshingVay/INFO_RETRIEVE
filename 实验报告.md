# 信息检索系统实验报告

## 一、项目概述

本项目设计并实现了一个基于**向量空间模型（Vector Space Model, VSM）**的信息检索系统，支持中文文档的索引构建与自然语言查询检索。系统包含网络爬虫、文本预处理、倒排索引构建、TF-IDF权重计算、余弦相似度匹配、结果排序展示以及检索结果人工评价等完整功能模块。

**技术栈**：Python 3、jieba分词、BeautifulSoup4网页解析、JSON本地存储

## 三、核心算法设计

### 3.1 倒排索引

对于每篇文档d，经过分词得到词项序列{t₁, t₂, ..., tₙ}，构建词项到文档的映射：

```
倒排索引结构:
{
  "词项1": {doc_id1: tf, doc_id2: tf, ...},
  "词项2": {doc_id3: tf, doc_id4: tf, ...},
  ...
}
```

其中TF（词频）记录该词项在文档中出现的次数。同时计算每个词项的IDF（逆文档频率）：

$$IDF(t) = \log_{10}\left(\frac{N}{df(t) + 1}\right) + 1$$

其中N为文档总数，df(t)为包含词项t的文档数。

### 3.2 向量空间模型（VSM）

**TF-IDF权重计算**：

$$w_{t,d} = \frac{tf_{t,d}}{|d|} \times IDF(t)$$

其中|d|为文档d的词项总数，对TF进行文档长度归一化。

**查询向量化**：对用户输入的自然语言查询进行分词后，同样计算TF-IDF权重得到查询向量。

**余弦相似度**：计算查询向量与文档向量之间的夹角余弦值：

$$sim(q, d) = \frac{\sum_{t \in q \cap d} w_{t,q} \cdot w_{t,d}}{\sqrt{\sum_t w_{t,q}^2} \cdot \sqrt{\sum_t w_{t,d}^2}}$$

按相似度从大到小排序，返回Top-K篇最相关文档。

### 3.3 文本预处理

1. **HTML清洗**：去除HTML标签、特殊字符，统一空白字符
2. **中文分词**：使用jieba分词库进行中文分词（精确模式）
3. **停用词过滤**：内置200+中文停用词（的、了、在、是...），过滤无意义词项
4. **数字过滤**：过滤纯数字词项
5. **单字符过滤**：过滤非中文单字符

## 六、人工评价系统

系统提供了交互式检索结果评价功能，支持：
1. 预设10组评测查询（覆盖AI、5G、新能源、芯片、大数据、航天、环保、医疗、区块链、机器人等方向）
2. 用户查看每个查询的Top-10检索结果
3. 用户标注相关文档编号
4. 自动计算并展示各查询的**Precision@10**（准确率）和**Recall**（召回率）
5. 汇总所有查询的平均准确率和平均召回率

## 七、创新点与优化

### 7.1 多源智能爬虫
- 支持人民网、新华网、中国日报三大中文新闻源
- 自适应编码检测（apparent_encoding）
- 随机User-Agent轮换，设置请求延迟防封
- 断点续爬：自动加载已有文档，去重URL
- 包含100+样本文档作为兜底，确保系统可用性

### 7.2 高效索引结构
- 使用Python defaultdict实现稀疏倒排索引
- IDF值预计算并缓存，加速检索
- 索引持久化为JSON，支持加载复用

### 7.3 智能摘要生成
- 定位查询词项在文档中的位置
- 以匹配位置为中心截取上下文窗口
- 按句子边界切分，保证摘要可读性

### 7.4 完整评价体系
- 支持Precision@K和Recall双指标评价
- 评价结果持久化，支持增量标注
- 预设10组评测查询覆盖不同领域

## 八、环境与社会可持续发展考虑

1. **绿色计算**：倒排索引结构高效紧凑，115篇文档的索引仅占86KB，检索时间复杂度O(|Q|×avg_postings)，空间和时间效率优秀
2. **本地存储**：所有文档和索引本地化存储，减少网络依赖，降低能耗
3. **断点续爬**：避免重复爬取已获取的文档，减少不必要的网络请求
4. **模块化设计**：各模块松耦合，易于扩展和维护，降低软件生命周期中的资源消耗
5. **轻量级实现**：使用纯Python实现，无需GPU等高性能计算资源，可在普通硬件上运行
6. **样本文档兜底**：减少对网络爬虫的依赖，降低对目标网站的请求压力

## 十、运行说明

### 环境要求
- Python 3.8+
- 依赖：requests, beautifulsoup4, jieba, lxml

### 安装与运行
```bash
# 安装依赖
pip install requests beautifulsoup4 jieba lxml

# 运行系统
python main.py
```

### 操作流程
1. 选择 `[1]` 爬取文档（首次使用）
2. 选择 `[2]` 构建倒排索引
3. 选择 `[3]` 输入查询关键词搜索
4. 选择 `[4]` 进入交互式查询模式
5. 选择 `[5]` 进行人工评价
6. 选择 `[6]` 查看系统状态

## 十一、总结

本项目成功实现了一个功能完整的中文信息检索系统，涵盖文档采集、分词预处理、倒排索引构建、向量空间模型检索、结果排序展示和人工评价等完整流程。系统共索引115篇中文文档、642个词项，检索响应迅速，结果准确。通过人工评价模块，可以对检索效果进行量化评估。系统采用模块化设计，具有良好的可扩展性。
